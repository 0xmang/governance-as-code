# Site Summary

## Project title

Multi-Cloud Governance-as-Code Lab: AWS + Azure + Terraform + ISO 27001 Mapping

## Short summary

I built a multi-cloud Governance-as-Code lab using Terraform to demonstrate how security and compliance requirements can be translated into repeatable cloud controls. The project deploys a governance baseline across AWS and Azure, including policy enforcement, configuration monitoring, audit logging, and ISO 27001-aligned control mapping.

## Medium summary

This project showcases a practical Governance-as-Code approach across AWS and Azure. Using Terraform, I implemented a baseline of logging, configuration monitoring, policy enforcement, and secure storage controls to demonstrate how cloud governance can move from documents and checklists into codified, repeatable guardrails. I also mapped the implemented controls to ISO/IEC 27001:2022 themes to show how cloud-native services can support broader governance objectives.
